# 3. Tipos MIME

## Objetivos
- Comprender la función de los tipos MIME.

### 3.1 ¿Qué es un tipo MIME?
Identifica el tipo de contenido.

### 3.2 Ejemplos
text/html, text/css, image/jpeg, application/json

### 3.3 Importancia
Permite interpretar correctamente contenido.

### 3.4 Configuración Apache
```
AddType text/html .html
```

### 3.5 Verificación
Herramientas F12 del navegador.
