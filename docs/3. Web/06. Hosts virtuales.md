# 6. Hosts virtuales

## Objetivos
- Gestionar múltiples sitios web.

### 6.1 Concepto
Un servidor puede alojar varios sitios.

### 6.2 Tipos
Basados en nombre o IP.

### 6.3 Ejemplo
```
ServerName ejemplo.com
DocumentRoot /var/www/ejemplo.com
```

### 6.4 Activación
```
sudo a2ensite ejemplo.com.conf
```

### 6.5 Múltiples sitios
Un archivo por dominio.
