# 7. Autentificación y control de acceso

## Objetivos
- Proteger recursos web.

### 7.1 Concepto
Verificar identidad.

### 7.2 Métodos
.htpasswd, LDAP, bases de datos.

### 7.3 Ejemplo básico
```
AuthType Basic
Require valid-user
```

### 7.4 Grupos
```
Require group admin
```

### 7.5 IP
```
Require ip 192.168.1.0/24
```
