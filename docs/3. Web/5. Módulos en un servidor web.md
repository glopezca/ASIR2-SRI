# 5. Módulos en un servidor web

## Objetivos
- Entender módulos del servidor.

### 5.1 ¿Qué son?
Extensiones funcionales.

### 5.2 Instalación
```
sudo apt install libapache2-mod-ssl
```

### 5.3 Activación
```
sudo a2enmod ssl
```

### 5.4 mod_rewrite
```
sudo a2enmod rewrite
```

### 5.5 Módulos esenciales
mod_ssl, mod_rewrite, mod_headers, mod_proxy, mod_security, mod_userdir
