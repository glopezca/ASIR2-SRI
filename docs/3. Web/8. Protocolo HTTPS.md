# 8. Protocolo HTTPS

## Objetivos
- Configurar HTTPS.

### 8.1 ¿Qué es?
HTTP con SSL/TLS.

### 8.2 Importancia
Seguridad y confianza.

### 8.3 Configuración
```
SSLEngine on
SSLCertificateFile /etc/ssl/certs/site.crt
```

### 8.4 Autofirmado
```
openssl req -x509 ...
```

### 8.5 Let's Encrypt
```
sudo certbot --apache
```

### 8.6 Redirección
```
Redirect / https://dominio.com/
```
