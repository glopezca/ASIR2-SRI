# 2. Protocolo HTTP

## Objetivos
- Comprender el funcionamiento y características del protocolo HTTP.
- Explicar solicitudes y respuestas HTTP.

## Contenidos

### 2.1 ¿Qué es HTTP?
HTTP es un protocolo de la capa de aplicación para comunicación entre clientes y servidores web.

### 2.2 Solicitudes HTTP
Incluye línea de solicitud, encabezados y cuerpo.

### 2.3 Respuestas HTTP
Incluye código de estado, encabezados y cuerpo.

### 2.4 Ejemplo
```
GET /index.html HTTP/1.1
Host: www.ejemplo.com
```

### 2.5 Métodos comunes
GET, POST, PUT, DELETE

### 2.6 Códigos
1xx–5xx
