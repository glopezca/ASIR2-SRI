# 1. Introducción y Características Generales de un Servidor Web

## Objetivos

-   Familiarizar a los estudiantes con el concepto de servidor web, sus
    características básicas y sus aplicaciones.\
-   Presentar los tipos principales de servidores web y sus funciones en
    una infraestructura de red.

## Contenidos

### 1.1 ¿Qué es un Servidor Web?

Un servidor web es una aplicación o programa que se ejecuta en un equipo
o servidor físico y proporciona contenido web a través de la red.\
Su función principal es gestionar las solicitudes de los usuarios y
enviarles los archivos o respuestas solicitadas, que pueden incluir:

-   Páginas web (HTML, CSS, JavaScript)\
-   Archivos multimedia (imágenes, vídeos, documentos)\
-   Respuestas de aplicaciones dinámicas (PHP, Python, JavaScript en
    servidor)

**Principio básico de funcionamiento:** 1. Un cliente (generalmente un
navegador web) realiza una solicitud al servidor.\
2. El servidor web recibe y procesa la solicitud, enviando la respuesta
si el recurso solicitado está disponible.\
3. El cliente recibe los datos y los muestra al usuario en el navegador.

------------------------------------------------------------------------

### 1.2 Tipos de Servidores Web

Existen distintos tipos de servidores web, cada uno con características
específicas que los hacen adecuados para determinados entornos:

-   **Apache HTTP Server**\
    Popular y ampliamente utilizado, especialmente en sistemas Linux.
    Destaca por su flexibilidad y su sistema de módulos.

-   **Nginx**\
    Enfocado en la eficiencia y el rendimiento, especialmente en
    conexiones concurrentes. Suele utilizarse en sitios de alto tráfico.

-   **Microsoft IIS**\
    Servidor web de Microsoft, integrado con ASP.NET y otras tecnologías
    del ecosistema Windows.

------------------------------------------------------------------------

### 1.3 Componentes Básicos de un Servidor Web

Un servidor web funcional debe incluir:

-   **Sistema operativo:** Gestiona los recursos del hardware y permite
    la comunicación en red.\
-   **Software de servidor web:** Como Apache, Nginx o IIS.\
-   **Directorio de contenido:** Espacio donde se almacenan los archivos
    servidos al cliente.\
-   **Registros (logs):** Archivos con información sobre el tráfico y la
    actividad del servidor.

------------------------------------------------------------------------

### 1.4 Ejemplo práctico: Instalación de Apache en Ubuntu

#### Instalación del servidor Apache

``` bash
sudo apt update
sudo apt install apache2
```

#### Verificar el estado del servicio

``` bash
sudo systemctl status apache2
```

#### Probar el servidor

Abrir en el navegador:

    http://localhost

#### Crear una página de prueba

``` bash
echo "<h1>Bienvenidos a nuestro servidor web en Apache</h1>" | sudo tee /var/www/html/index.html
```

Este primer ejercicio permite a los estudiantes relacionar la teoría con
la práctica mediante la instalación y verificación de un servidor web
real.
