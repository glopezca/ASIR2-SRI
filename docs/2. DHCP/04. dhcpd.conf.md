# 4. dhcpd.conf

## Objetivos
- Entender la estructura del archivo de configuración de ISC DHCP.
- Aprender a definir rangos, parámetros globales y reservas.

## 4.1 Ubicación del archivo
El archivo suele encontrarse en:

```
/etc/dhcp/dhcpd.conf
```

## 4.2 Ejemplo básico
```
default-lease-time 600;
max-lease-time 7200;

subnet 192.168.1.0 netmask 255.255.255.0 {
    range 192.168.1.100 192.168.1.200;
    option routers 192.168.1.1;
    option domain-name-servers 192.168.1.10, 192.168.1.11;
    option domain-name "asir2.fp";

    host cliente1 {
        hardware ethernet 00:11:22:33:44:55;
        fixed-address 192.168.1.50;
    }
}
```

## 4.3 Explicación línea a línea
- **default-lease-time**: tiempo por defecto para clientes.
- **range**: rango de IP asignables.
- **option routers**: gateway predeterminado.
- **host clienteX**: reservas estáticas.
