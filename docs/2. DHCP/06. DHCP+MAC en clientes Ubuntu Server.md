# 6. DHCP+MAC en clientes Ubuntu Server

## Objetivos
- Configurar Netplan para usar la MAC como identificador DHCP.
- Forzar identificación mediante MAC en Ubuntu Server.

## 6.1 Configuración Netplan

Editar archivo `/etc/netplan/*.yaml`:

```yaml
network:
  version: 2
  ethernets:
    eth0:
      dhcp4: true
      dhcp-identifier: mac
```

Aplicar cambios:
```
sudo netplan apply
```

## Valores permitidos para dhcp-identifier
- **duid**
- **mac**
- **vendor**
