# 8.1 NAT en Ubuntu

## Objetivos
- Configurar NAT de sobrecarga (masquerading) en Linux.

## 9.1 Habilitar reenvÃ­o de paquetes
```
sudo sysctl -w net.ipv4.ip_forward=1
```

## 9.2 Regla NAT con iptables
```
sudo iptables -t nat -A POSTROUTING -o eth0 -j MASQUERADE
```

## 9.3 Guardar reglas
```
sudo iptables-save > /etc/iptables/rules.v4
```
