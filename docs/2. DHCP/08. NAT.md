# 8. NAT

## Objetivos
- Comprender la traducción de direcciones de red.
- Distinguir tipos de NAT.

## 8.1 ¿Qué es NAT?
Permite que múltiples dispositivos utilicen una única IP pública.

## 8.2 Tipos de NAT

| Tipo | Descripción | Ventajas | Desventajas |
|------|-------------|----------|-------------|
| Estático | IP fija pública ↔ privada | Accesible desde fuera | Requiere muchas IPs |
| Dinámico | Asignación desde un pool | Flexible | Requiere varias IPs |
| PAT | Una IP pública con puertos | Muy eficiente | Limitado por puertos |
