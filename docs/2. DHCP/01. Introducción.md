# 1. Introducción

## Objetivos
- Introducir el protocolo DHCP.
- Comprender su función en la asignación automática de parámetros de red.
- Explicar el concepto de "tiempo de arrendamiento" (lease) y su utilidad.

## 1.1 Introducción al Protocolo DHCP
DHCP (Dynamic Host Configuration Protocol) es un protocolo de red que permite asignar automáticamente configuraciones de red a los dispositivos que se conectan a una red, como:

- Dirección IP  
- Máscara de subred  
- Puerta de enlace predeterminada  
- Servidores DNS  

Gracias a DHCP, la administración de direcciones IP se simplifica enormemente, evitando configuraciones manuales y reduciendo conflictos de IP.

## 1.2 Funcionamiento general
Cuando un dispositivo se une a la red:
1. Solicita configuración.
2. El servidor DHCP responde con los parámetros adecuados.
3. Se asigna un "tiempo de arrendamiento" (lease) tras el cual la IP debe renovarse.
