# 2. Funcionamiento

## Objetivos
- Comprender el proceso DORA.
- Describir los mensajes DHCP implicados en la asignación de IP.

## 2.1 Funcionamiento del protocolo DHCP
DHCP funciona mediante un proceso de cuatro fases llamado **DORA**:

### 1. Discover  
El cliente envía un mensaje **DHCPDISCOVER** para localizar servidores DHCP disponibles.

### 2. Offer  
El servidor responde con **DHCPOFFER**, ofreciendo una dirección IP y parámetros adicionales.

### 3. Request  
El cliente selecciona una oferta y responde con **DHCPREQUEST**, confirmando qué servidor ha elegido.

### 4. Acknowledgment  
El servidor confirma la asignación mediante **DHCPACK**, otorgando la IP y configuraciones finales.

Este proceso garantiza una asignación ordenada y evita conflictos de dirección IP.
