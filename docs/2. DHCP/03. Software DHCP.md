# 3. Software DHCP

## Objetivos
- Comparar distintos servidores DHCP.
- Identificar ventajas y limitaciones de ISC DHCP, Kea DHCP y dnsmasq.

## 3.1 Comparación de Servidores DHCP

| Característica | ISC DHCP | Kea DHCP | dnsmasq |
|---------------|----------|----------|---------|
| Desarrollo | Maduro, estable | Reemplazo moderno, API REST | Ligero, uso doméstico |
| Soporte IPv6 | Complejo | Nativo y robusto | Básico |
| Escalabilidad | Alta | Muy alta | Baja |
| Configuración | Compleja (dhcpd.conf) | Flexible | Simple |
| Seguridad | Básica | Mejorada | Limitada |
| Uso típico | Redes grandes | Grandes organizaciones | Redes domésticas |

### 3.2 Asignación estática
Todos los sistemas permiten reservas basadas en identificadores como MAC o DUID.
